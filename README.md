# HikeLog
OSS Hike Log web app

This is hobby project for logging hikes and travels. 
Web page of project is at [hikelog.org](https://hikelog.org), 
real application is at [zuzkajde.cz](https://zuzkajde.cz).

## Chosen technologies
I chose these technologies not because they are the best for the job, but because I wanted to learn them:
* FastAPI backend
* React frontend
* TailwindCSS for styling
* Docker for containerization
* Postgres database
